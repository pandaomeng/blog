# npm 相关命令记录

## 查看全局安装过的包

```
npm list -g --depth 0
```

参数解释

| 命令                      | 解释             |
| :------------------------ | ---------------- |
| npm list                  | 显示安装过的包   |
| -g                        | 指全局安装过的包 |
| --depth 0限制输出模块层级 |                  |
