---
title: Mac Sublime 快捷键
date: 2018-06-07
tag: [sublime, hotkey]
---

| 按键         | 命令                                            |
| ------------ | ----------------------------------------------- |
| ⌘ + ⇧ + P    | 命令提示（非常常用）                            |
| ⌘ + K, ⌘ + B | 侧边栏打开/关闭切换                             |
| ⌘ + ⌥ + 2    | 拆分窗口，双列显示                              |
| ⌘ + D        | 选择词 – 重复按下时多重选择相同的词进行多重编辑 |
| ⌘ + ⌃ + ↑    | 向上移动当前行/所选行                           |
| ⌘ + ⇧ + F    | 在所有打开的文件中进行查找                      |
| ⌘ + P        | 通过文件名快速打开                              |
